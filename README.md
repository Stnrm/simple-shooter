# simple-shooter

First-year project. 
Simple two-player space shooter game built with PyGame. 

To run you need PyGame.

## First player controls:
- W - Up
- D - Right
- A - Left
- S - Back
- G - Shoot

## Second player controls:
- Arrow-up - Up
- Arrow-right - Right
- Arrow-left - Left
- Arrow-down - Back
- Space - Shoot
